# GR86 Session Logger

## Project Description
The GR86 Session Logger is a comprehensive application designed to log and analyze driving sessions for the Toyota GR86 vehicle. This project aims to enhance the driving experience by providing detailed insights into performance metrics, session data, and the overall health of the vehicle. 

## Features
- **Real-Time Data Logging**: Capture various metrics during driving, such as speed, RPM, throttle position, and more.
- **Session Management**: Organize and manage multiple driving sessions efficiently.
- **Data Visualization**: Display session data through charts and graphs for better analysis.
- **Export Functionality**: Ability to export session data in various formats for further analysis or sharing.
- **User-friendly Interface**: A simple and intuitive interface to facilitate easy navigation and operation.
